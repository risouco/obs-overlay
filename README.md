# OBS オールインワン オーバーレイ

配信用のオーバーレイツール。時計・タイマー・カウンター・告知ティッカー・画像スワイプロールを1画面で管理できる、HTML1ファイル完結のツールです。

## 使い方（OBS）
1. OBSで **ソース → ＋ → ブラウザ** を追加
2. 公開URL `https://risouco.github.io/obs-overlay/obs-overlay.html` を指定（またはローカルの `obs-overlay.html`）
3. 幅 **1920** × 高さ **1080**、「表示されていないときにシャットダウン」は **OFF** 推奨
4. 配置後、右上「配信モードにする」でパネルと枠を隠す
5. 配信中の操作はソース右クリック →「対話（Interact）」から

## 画像スワイプロール用の画像
`images/` フォルダに入れて、コントロールパネルで `images/名前.png` と指定してください。

## 設定
各PCのブラウザ内に自動保存されます。設定の書き出し/読み込みは全体タブから。
