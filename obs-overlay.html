使い方：OBSで「ブラウザソース」を追加 → このファイルを指定（幅1920×高さ1080・「透過を許可」ON）。
配信時は右上の「配信モードにする」を押すとパネルと枠が消えて、オーバーレイだけになります。
各パーツは編集モード中にドラッグで自由に配置できます。設定はこのPCに自動保存されます。
複数行にすると「◆」区切りで順番に流れます。
「images」フォルダ（このファイルと同じ場所）に画像を入れて images/名前.png と書くのが軽くておすすめ。
「PCから画像を選ぶ」でも追加できますが、大きい画像を大量に入れると保存容量を超えることがあります。
自動取得（YouTube）：登録者数（YouTube仕様で概数）・高評価数（正確）・同時視聴者数。約60秒ごとに更新。
　→ APIキーは Google Cloud で無料発行 → 「YouTube Data API v3」を有効化 → キーに HTTPリファラー制限 で risouco.github.io/* を設定すると安全。
　チャンネルID（登録者用）は UCxxxx…、動画ID（高評価・同接用）は配信URLの v= 以降。
手動：Twitch・ニコ生など自動非対応の媒体用。＋/−や直接入力で更新。媒体が変わったら項目の種類を切り替えればOK。
